TempoLink

초저지연 실시간 오디오 잼 세션. 웹과 데스크탑을 아우르는 최고의 온라인 합주 플랫폼을 지금 경험해 보세요.

최신 릴리즈 버전: v0.1.0
🪟

Windows

Windows 10 / 11 64-bit 환경. ASIO 드라이버 및 필수 런타임이 함께 설치됩니다.

Windows 설치 다운로드
자동 환경 구성을 포함한 Installer (exe)
🍎

macOS

Intel & Apple Silicon (M1/M2/M3) 기본 지원. 가상 오디오 드라이버 구성이 포함됩니다.

macOS 설치 다운로드
CoreAudio 호환 Installer (pkg)
🐧

Ubuntu Linux

Ubuntu 20.04 LTS 이상 환경. ALSA/JACK 통합 설정과 관련 패키지가 포함됩니다.

Ubuntu 설치 다운로드
의존성 패키지 관리형 Installer (deb)
